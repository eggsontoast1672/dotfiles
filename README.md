# dotfiles

A bunch of configurations for apps that I use a lot.

I'm a distrohopper, so it can get a litte bit annoying when I have to redo my
configurations over and over again. It can get especially annoying with
programs like alacritty, neovim, and i3 where I like to tweak the config files
a ton.

I thought I would make a public repository for them just in case I ever become
famous one day and everyone wants my configurations for some reason.

Enjoy!
